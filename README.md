# Internshala Internship Listing Clone

A clone of the Internshala internship listings page built with **Next.js 15**, **TypeScript**, and **Tailwind CSS** as a college assignment.

## Features

- Browse internships from top Indian companies
- Filter by profile, location, duration, and minimum stipend
- Toggle switches for Work From Home and Part-time
- Global search across title, company, profile, and location
- Sort by relevance, stipend, duration, or latest
- Active filter chips with one-click removal
- Responsive design with mobile filter drawer
- Staggered card animations and skeleton shimmer loading
- Premium, PPO, WFH, Part-time, and Immediate label badges
- Optimized images using Next.js `<Image>` component
- Font preloading with preconnect for faster page loads

## Tech Stack

- **Framework:** Next.js 15 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS v4
- **Fonts:** Plus Jakarta Sans (body), Sora (headings)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
```
